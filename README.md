# Euro/Lev Currency Converter Firefox Extension

![Extension Icon](icons/icon128.png)

## Description
A Firefox extension for quick conversion between Euro (EUR) and Bulgarian Lev (BGN) using the fixed exchange rate of the Bulgarian National Bank (1 EUR = 1.95583 BGN).

## Features
- Real-time conversion between EUR and BGN
- Clean interface with smooth animations
- Swap currencies with one click
- Copy result to clipboard
- Save state between sessions
- Works offline

## Installation
1. Install from [Firefox Add-ons Store](https://addons.mozilla.org) (link will be added after approval)
2. Or load temporarily from `about:debugging`
